# Проект: "ИИ-продавец для EdTech"

## Описание проекта

Проект "ИИ-продавец для EdTech" представляет собой инновационное решение, направленное на автоматизацию процесса взаимодействия с потенциальными клиентами в сфере образовательных технологий (EdTech). Основная цель проекта — создание интеллектуального ассистента, который сможет эффективно отвечать на вопросы пользователей, предоставлять информацию о продуктах и услугах, а также организовывать встречи в автоматическом режиме.

Используя передовые технологии искусственного интеллекта, проект обеспечивает высокий уровень персонализации и удобства для пользователей, что способствует повышению конверсии и улучшению качества обслуживания.

---

## Функционал

### 1. **Осмысленное ведение диалога**
   - ИИ-ассистент способен поддерживать естественный диалог с пользователями, используя контекст и предыдущие сообщения.
   - Понимание намерений клиента и предоставление релевантных ответов.
   - Адаптация стиля общения в зависимости от запроса пользователя (формальный или неформальный тон).

### 2. **Ответы на вопросы по базе знаний**
   - Ассистент имеет доступ к структурированной базе знаний, содержащей информацию о продуктах, услугах, акциях и других аспектах компании.
   - Возможность быстро находить и предоставлять точные ответы на часто задаваемые вопросы.
   - Автоматическое обновление базы знаний для поддержания актуальности информации.

### 3. **Запись клиента на встречу**
   - Интеграция с Google Календарь для автоматической записи клиентов на встречи.
   - Умение согласовывать удобное время для встречи, учитывая занятость менеджеров и предпочтения клиента.
   - Отправка напоминаний о предстоящих встречах через электронную почту или мессенджеры.

---

## Используемые сервисы

### 1. **Qwen**
   - Применяется для создания системного промпта, который определяет логику и стиль работы ИИ-ассистента.
   - Разработка и поддержка базы знаний, обеспечивающей высокую точность ответов на вопросы пользователей.
   - Настройка модели для осмысленного ведения диалога с учетом специфики EdTech-сферы.

### 2. **Савви (Suvvy.ai)**
   - Платформа для создания и развертывания ИИ-ассистента.
   - Предоставляет инструменты для интеграции с внешними сервисами, такими как Google Календарь.
   - Обеспечивает удобный интерфейс для управления и мониторинга работы ассистента.

---

## Преимущества проекта

- **Автоматизация рутинных задач:** Освобождение сотрудников от необходимости отвечать на однотипные вопросы и организовывать встречи.
- **Повышение конверсии:** Быстрое и качественное обслуживание клиентов увеличивает вероятность их перехода на следующие этапы воронки продаж.
- **Масштабируемость:** Решение легко адаптируется под потребности компаний любого размера.
- **Круглосуточная доступность:** ИИ-ассистент работает 24/7, что позволяет обслуживать клиентов в любое время суток.

---

## План внедрения

1. **Анализ требований:** Сбор информации о продуктах, услугах и целевой аудитории компании.
2. **Создание базы знаний:** Наполнение базы знаний актуальной информацией с использованием Qwen.
3. **Настройка диалогового ассистента:** Разработка промптов и тестирование функционала ведения диалога.
4. **Интеграция с Google Календарь:** Настройка автоматической записи на встречи.
5. **Тестирование:** Проведение тестовых сценариев для выявления и устранения возможных ошибок.
6. **Запуск и поддержка:** Внедрение решения в рабочий процесс компании и последующая поддержка.

---

## Заключение

Проект "ИИ-продавец для EdTech" предлагает комплексное решение для автоматизации взаимодействия с клиентами в образовательной сфере. Благодаря использованию современных технологий искусственного интеллекта, проект обеспечивает высокий уровень удовлетворенности клиентов и помогает компаниям достигать своих бизнес-целей более эффективно.
